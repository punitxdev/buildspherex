> ⚠️ **Development Status:** BuildSphereX is a core initiative driving **startup culture** across our campus! Please note that the platform is currently under active development and building.

## 🌟 The Vision
The university ecosystem is filled with brilliant minds, but finding the right co-founders, discussing technical roadblocks, and securing funding or incubation can be incredibly fragmented. **BuildSphereX** bridges that gap by providing a localized, centralized hub where students can pitch ideas, form teams, and bring their innovations to life inside the campus!

## ✨ Key Features
- **🔐 Secure Authentication:** Complete JWT-based user login and registration tailored for students.
- **💡 Idea Boards:** Pitch your latest project, tag required skills (e.g., MERN stack, Embedded Systems), and recruit collaborators from other departments.
- **🗣️ Discussion Forums:** Ask technical questions about frameworks, discuss hardware constraints, or share tips on pitching to campus incubators.
- **👤 Dynamic Profiles:** Showcase your skills, your ongoing projects, and your academic background.
- **📢 Real-Time Announcements:** Stay updated on hackathons, seed funding grants, and tech meets.

## 🛠️ Tech Stack
BuildSphereX is proudly built on the **MERN** stack for scalable, rapid development:
- **Frontend:** React.js (powered by Vite) + Context API for fast rendering.
- **Backend:** Node.js & Express.js for building robust REST APIs.
- **Database:** MongoDB & Mongoose for flexible, document-based data storage.
- **Styling:** Custom Vanilla CSS for a premium glassmorphic UI.

## 🚀 Quick Start
To run BuildSphereX locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/punitxdev/buildspherex.git
   cd buildspherex
   ```
2. **Start the Express API (Backend):**
   ```bash
   cd server
   npm install
   npm run dev
   ```
3. **Start the Vite UI (Frontend):**
   *(Note: Vite requires Node v20 or higher)*
   ```bash
   # In a new terminal tab at the project root
   npm install
   npm run dev
   ```
