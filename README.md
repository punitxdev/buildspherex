# BuildSphereX

BuildSphereX is a comprehensive student startup networking and collaboration platform. 

**Note: It is an initiative for the startup culture for the campus and it is currently under building/development.**

## Features
- Complete Authentication System
- Idea creation and collaboration boards
- Discussion forums and community engagement
- User profiles

## Technologies Used
- MongoDB
- Express.js
- React (Vite)
- Node.js
